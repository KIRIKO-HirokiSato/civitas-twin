# Civitas Twin 動作テスト用入力

以下をそのままフォームにコピー＆ペーストしてシミュレーションを実行してください。

## テストケース1：施策・サービス（ややユニーク）

**解析対象（ニュース/施策）**
```
2026年度から、全国の公営住宅で「ペット可」物件の割合を現行の5%から20%に段階的に引き上げる方針が国土交通省より示された。入居希望者のニーズ調査で「ペットと暮らしたい」が若年層で急増していることを受けた措置。犬・猫に加え、一定条件でウサギ・ハムスターも対象とする見込み。
```

**目的 / 対象者 / 条件**
```
公営住宅入居希望者および既存入居者への周知方針の検討。賛否両論が予想されるため、年齢・居住形態別の受容度を把握したい。
```

**地域**
```
日本全体（都市部・郊外・地方の比較を想定）
```

---

## テストケース2：事象・ニュース（ユニーク）

**解析対象（ニュース/施策）**
```
某県が「午前中の会議禁止条例」を全国で初めて検討していると報じられた。午前は集中・創造的作業に充て、会議は午後からに統一することで生産性と働き方満足度の向上を目指す。企業への強制ではなく、県民・県内事業者への推奨という位置づけ。
```

**目的 / 対象者 / 条件**
```
条例案の県民理解度と、企業・労働者それぞれの反応予測。賛成層・反対層・無関心層の割合と理由の傾向を知りたい。
```

**地域**
```
当該県および隣接県（波及効果の有無を想定）
```

---

## テストケース3：広報メッセージ（短め・コピペ用）

**解析対象**
```
当社は来春、サブスク型「お惣菜デリバリー」を首都圏で開始します。毎週決まった曜日に冷蔵便で届き、電子レンジで温めるだけ。価格は1食あたり税込450円〜。忙しい共働き世帯と単身者を主ターゲットにしています。
```

**目的**
```
リリース前のメッセージテスト。価格・ターゲット設定に対する社会層別の反応をシミュレートしたい。
```

**地域**
```
首都圏（東京都・神奈川・埼玉・千葉）
```
