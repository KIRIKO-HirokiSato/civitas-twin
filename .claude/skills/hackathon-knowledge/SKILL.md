# 第4回 Agentic AI Hackathon with Google Cloud

Google Cloud Japan協賛、Zenn主催のAIエージェント開発ハッカソン。
賞金総額175万円。テーマ: 「進化するAIと共に。君だけの『エージェント』を創り出そう」

## クイックリファレンス

| 項目 | 内容 |
|------|------|
| 提出締切 | **2026年2月15日** |
| 1次審査 | 2026年2月16日〜2月23日 |
| 2次審査 | 2026年2月24日〜3月2日 |
| 最終審査 | 2026年3月19日（Google Cloud Agentic AI Summit内） |
| 最優秀賞 | 50万円（1件） |
| 優秀賞 | 25万円（3件） |
| 奨励賞 | 10万円（5件） |

## 必須技術要件（両方満たす必要あり）

1. **Google Cloud実行プロダクト** 1つ以上:
   App Engine / Compute Engine / GKE / Cloud Run / Cloud Functions / TPU・GPU

2. **Google Cloud AI技術** 1つ以上:
   Vertex AI / Gemini API / Gemma / Imagen / Agent Builder / ADK 等

## 提出物（3点セット）

1. 公開GitHubリポジトリURL
2. デプロイ済みプロジェクトURL
3. Zenn記事（カテゴリ「Idea」、トピック「gch4」）
   - プロジェクト概要
   - システムアーキテクチャ図
   - 約3分のデモ動画（YouTube埋め込み）

## 審査基準

1. **課題の新規性** — 未解決課題の発見度合い
2. **解決策の有効性** — 課題への対処と解決の効果性
3. **実装品質と拡張性** — アイデア実現度、拡張性、運用性、費用対効果
