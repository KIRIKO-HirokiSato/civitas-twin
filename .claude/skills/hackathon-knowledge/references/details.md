# ハッカソン詳細情報

## 基本情報

- **正式名称**: 第4回 Agentic AI Hackathon with Google Cloud
- **期間**: 2025年12月10日〜2026年2月15日
- **賞金総額**: 175万円
- **主催**: Zenn（クラスメソッド株式会社）
- **協賛**: グーグル・クラウド・ジャパン合同会社
- **公式ページ**: https://zenn.dev/hackathons/google-cloud-japan-ai-hackathon-vol4

## 参加条件

- 日本国内居住の18歳以上の個人またはチーム
- **除外対象**: 政府機関職員、企業代表として選出された者、主催者・スポンサー関係者、運営に直接関わる者

## 技術要件（詳細）

### Google Cloud実行プロダクト（1つ以上必須）

| プロダクト | 用途例 |
|-----------|--------|
| App Engine | Webアプリホスティング |
| Compute Engine | VM上でのカスタム処理 |
| GKE | コンテナオーケストレーション |
| Cloud Run | サーバーレスコンテナ |
| Cloud Functions | イベント駆動処理 |
| TPU/GPU | ML推論・学習ワークロード |

### Google Cloud AI技術（1つ以上必須）

| 技術 | 説明 |
|------|------|
| Vertex AI | MLプラットフォーム全般 |
| Gemini API | LLM API |
| Gemma | オープンモデル |
| Imagen | 画像生成 |
| Agent Builder | エージェント構築ツール |
| ADK (Agent Development Kit) | エージェント開発キット |

## 提出物要件（詳細）

### 1. GitHubリポジトリ
- **公開**リポジトリであること
- ソースコード一式を含む

### 2. デプロイ済みプロジェクトURL
- 実際にアクセス・動作確認が可能な状態

### 3. Zenn記事
- **カテゴリ**: Idea

- **必須コンテンツ**:
  - プロジェクト概要
  - システムアーキテクチャ図
  - 約3分のデモ動画（YouTube埋め込み）

## 審査基準（詳細）

### 1. 課題の新規性
- まだ解決されていない課題を見つけているか
- 既存のアプローチとの差別化があるか

### 2. 解決策の有効性
- 提示された課題に対して効果的に対処できているか
- 実際に課題を解決する力があるか

### 3. 実装品質と拡張性
- アイデアがどこまで実現されているか
- 将来的な拡張が可能な設計か
- 運用を見据えた実装か
- 費用対効果は妥当か
